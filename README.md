# Trash bin Finder

## Installation

Use this command in each service.

```bash
npm install
```

## QuickStart

start the service

```bash
npm start
```

testing

```bash
npm test
```
Make sure that these ports are not occupied

```bash
trash-bin-finder-service:3000
elasticsearch-dal:9000
tests:4444
```

## Notes

* Tests are located in trash-bin-finder-service
